# The Chimera Company

**The Doyenne of Events**

A luxury event management brand specializing in bespoke weddings, corporate events, and social celebrations.

## Brand Voice

- **Luxury**: Elevated, refined, sophisticated
- **Calm**: Confident, composed, unhurried  
- **Confident**: Authoritative, assured, expert
- **Human**: Warm, personal, authentic

Not salesy. Not generic. Every word serves the experience.

## Brand Positioning

**Tagline**: The Doyenne of Events

**Core Services**:
- Weddings
- Corporate Events  
- Social Events

**Value Proposition**: We design and deliver exceptional experiences for clients who value detail, discretion, and distinction. From intimate celebrations to grand destination events, every Chimera event is curated with purpose and executed with precision.

## Key Messaging

### Hero Statement
"Bespoke. Luxury. Destination Event Management."

### Value Statement  
"At The Chimera Company, we don't just plan events — we translate your story into an experience. Every detail is intentional. Every moment is considered. Every celebration feels effortless."

### Our Approach
Luxury lives in the details. Here's how we bring your vision to life:
1. **Discover** - We listen. Deeply. Your style, goals, and priorities shape everything.
2. **Design** - We curate concepts, vendors, and experiences that feel uniquely yours.
3. **Execute** - We manage logistics, timelines, and on-site coordination — flawlessly.
4. **Deliver** - You show up. We handle everything else.

### Call to Action
"Ready to Create Something Exceptional? Let's design an experience your guests will never forget."

- Next.js 15

- Framer Motion
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```

  about/               # About page
  services/            # Services page
  portfolio/           # Portfolio pages

  pages/               # Page-specific components
  shared/              # Reusable components

```
